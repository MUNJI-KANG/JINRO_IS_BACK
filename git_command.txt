✅ main -> 개인 브랜치로 풀 받기
git stash ➡ 자신이 작업 했던거 잠깐 숨기기
git pull origin main ➡ 최신 코드 받기
git stash pop ➡ 숨겼던 작업 다시 꺼내기
----------------------------------------------------------------
✅ main 최신화해야 할 때 이렇게 많이 함.
git stash
git switch main
git pull
git switch feature/AHV
git merge main
git stash pop
----------------------------------------------------------------
✅ 개인 브랜치 -> main 브랜치로 합치기
1. git switch "브랜치 이름"  ➡ 브랜치 이동
2. git pull origin main ➡ 최신 상태로 업데이트
3. git merge feature/"브랜치 이름" ➡ 해당 브랜치 merge
4. git push origin main ➡ 원격에 반영
----------------------------------------------------------------
git status ➡ 변경사항 확인
